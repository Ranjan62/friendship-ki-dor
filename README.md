# 🤝 Friendship Interactive UI

A fun, interactive web mini-project built using **HTML, CSS, and JavaScript**.
It presents a playful UI asking the user to accept friendship — with dynamic responses and animations.

---

## 🚀 Features

* 🎨 Modern **glassmorphism UI**
* 🌌 Gradient + starry background
* 😄 Interactive "Yes / No" decision flow
* ❌ "No" button:

  * Shows funny warning messages
  * Moves randomly to avoid clicks
  * Triggers shake animation
* ✅ "Yes" button:

  * Grows larger with each "No" click
  * Displays success screen with sound
* ✨ Smooth animations (fade + shake)
* 🎯 Fully responsive and centered layout

---

## 🧠 Concept

This project demonstrates:

* Basic **DOM manipulation**
* Event handling in JavaScript
* CSS animations using `@keyframes`
* UI/UX tricks (like button movement & scaling)

It also explores **playful user psychology** by:

* Making rejection harder
* Making acceptance more attractive over time
